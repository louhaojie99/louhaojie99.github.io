еҚҒе…ӯгҖҒJSONе’Ңlocalstorage

еҸ‘иЎЁдәҺ2018-01-16|жӣҙж–°дәҺ2021-03-23|JS
|йҳ…иҜ»йҮҸ:

JSON еӯ—з¬ҰдёІе’ҢеҜ№иұЎзҡ„иҪ¬жҚў

JSON(JavaScript Object Notation, JS еҜ№иұЎз®Җи°ұ) жҳҜдёҖз§ҚиҪ»йҮҸзә§зҡ„ж•°жҚ®дәӨжҚўж јејҸгҖӮе®ғеҹәдәҺ [ECMAScript] (ж¬§жҙІи®Ўз®—жңәеҚҸдјҡеҲ¶е®ҡзҡ„ js и§„иҢғ)зҡ„дёҖдёӘеӯҗйӣҶпјҢйҮҮз”Ёе®Ңе…ЁзӢ¬з«ӢдәҺзј–зЁӢиҜӯиЁҖзҡ„ж–Үжң¬ж јејҸжқҘеӯҳеӮЁе’ҢиЎЁзӨәж•°жҚ®гҖӮз®ҖжҙҒе’Ңжё…жҷ°зҡ„еұӮж¬Ўз»“жһ„дҪҝеҫ— JSON жҲҗдёәзҗҶжғізҡ„ж•°жҚ®дәӨжҚўиҜӯиЁҖгҖӮ жҳ“дәҺдәәйҳ…иҜ»е’Ңзј–еҶҷпјҢеҗҢж—¶д№ҹжҳ“дәҺжңәеҷЁи§Јжһҗе’Ңз”ҹжҲҗпјҢе№¶жңүж•Ҳең°жҸҗеҚҮзҪ‘з»ңдј иҫ“ж•ҲзҺҮ

жіЁж„Ҹ json еӯ—з¬ҰдёІдёӯзҡ„еұһжҖ§еҗҚеҸӘиғҪжҳҜеӯ—з¬ҰдёІпјҢе№¶дё”еҸӘиғҪз”ЁеҸҢеј•еҸ·гҖӮеұһжҖ§еҖјеҰӮжһңжҳҜеӯ—з¬ҰдёІзҡ„иҜқпјҢд№ҹеҸӘиғҪдҪҝз”ЁеҸҢеј•еҸ·

жҠҠ json еӯ—з¬ҰдёІиҪ¬дёә js дёӯзҡ„еҜ№иұЎ

жҠҠ js дёӯзҡ„еҜ№иұЎ z иҪ¬еҢ–дёә json еӯ—з¬ҰдёІ

JSON еёёи§ҒжҠҘй”ҷ

eval еҮҪж•°

eval() еҮҪж•°еҸҜи®Ўз®—жҹҗдёӘеӯ—з¬ҰдёІпјҢе№¶жү§иЎҢе…¶дёӯзҡ„зҡ„ JavaScript д»Јз ҒгҖӮ

localstorge

д»Җд№ҲжҳҜ localStorage

еңЁ HTML5 дёӯпјҢж–°еҠ е…ҘдәҶдёҖдёӘ localStorage зү№жҖ§пјҢиҝҷдёӘзү№жҖ§дё»иҰҒжҳҜз”ЁжқҘдҪңдёәжң¬ең°еӯҳеӮЁжқҘдҪҝз”Ёзҡ„пјҢ
и§ЈеҶідәҶ cookie еӯҳеӮЁз©әй—ҙдёҚи¶ізҡ„й—®йўҳ(cookie дёӯжҜҸжқЎ cookie зҡ„еӯҳеӮЁз©әй—ҙдёә 4k)пјҢ
localStorage дёӯдёҖиҲ¬жөҸи§ҲеҷЁж”ҜжҢҒзҡ„жҳҜ 5M еӨ§е°ҸпјҢиҝҷдёӘеңЁдёҚеҗҢзҡ„жөҸи§ҲеҷЁдёӯ localStorage дјҡжңүжүҖдёҚеҗҢ

localStorage зҡ„дјҳеҠҝ

1гҖҒlocalStorage жӢ“еұ•дәҶ cookie зҡ„ 4K йҷҗеҲ¶

2гҖҒlocalStorage дјҡеҸҜд»Ҙе°Ҷз¬¬дёҖж¬ЎиҜ·жұӮзҡ„ж•°жҚ®зӣҙжҺҘеӯҳеӮЁеҲ°жң¬ең°пјҢиҝҷдёӘзӣёеҪ“дәҺдёҖдёӘ 5M еӨ§е°Ҹзҡ„й’ҲеҜ№дәҺеүҚз«ҜйЎөйқўзҡ„ж•°жҚ®еә“пјҢзӣёжҜ”дәҺ cookie еҸҜд»ҘиҠӮзәҰеёҰе®ҪпјҢдҪҶжҳҜиҝҷдёӘеҚҙжҳҜеҸӘжңүеңЁй«ҳзүҲжң¬зҡ„жөҸи§ҲеҷЁдёӯжүҚж”ҜжҢҒзҡ„

localStorage зҡ„еұҖйҷҗ

1гҖҒжөҸи§ҲеҷЁзҡ„еӨ§е°ҸдёҚз»ҹдёҖпјҢе№¶дё”еңЁ IE8 д»ҘдёҠзҡ„ IE зүҲжң¬жүҚж”ҜжҢҒ localStorage иҝҷдёӘеұһжҖ§

2гҖҒзӣ®еүҚжүҖжңүзҡ„жөҸи§ҲеҷЁдёӯйғҪдјҡжҠҠ localStorage зҡ„еҖјзұ»еһӢйҷҗе®ҡдёә string зұ»еһӢпјҢиҝҷдёӘеңЁеҜ№жҲ‘д»¬ж—ҘеёёжҜ”иҫғеёёи§Ғзҡ„ JSON еҜ№иұЎзұ»еһӢйңҖиҰҒдёҖдәӣиҪ¬жҚў

localStorage зҡ„еҶҷе…Ҙ

localStorage еҸӘж”ҜжҢҒ string зұ»еһӢзҡ„еӯҳеӮЁ

        //еҶҷе…Ҙcеӯ—ж®ө
      localStorage.setItem("c",3);

localStorage зҡ„иҜ»еҸ–

        var c=localStorage.getItem("c");
        console.log(c)

localStorage зҡ„дҝ®ж”№
ж”№иҝҷдёӘжӯҘйӘӨжҜ”иҫғеҘҪзҗҶи§ЈпјҢжҖқи·Ҝи·ҹйҮҚж–°жӣҙж”№е…ЁеұҖеҸҳйҮҸзҡ„еҖјдёҖж ·

localStorage.setItem(вҖңcвҖқ,3);

е°Ҷ localStorage дёӯзҡ„жҹҗдёӘй”®еҖјеҜ№еҲ йҷӨ
localStorage.removeItem(вҖңaвҖқ);

е°Ҷ localStorage зҡ„жүҖжңүеҶ…е®№жё…йҷӨ

localStorage.clear();

localStorage е…¶д»–жіЁж„ҸдәӢйЎ№
дёҖиҲ¬жҲ‘д»¬дјҡе°Ҷ JSON(js дёӯзҡ„еҜ№иұЎ)еӯҳе…Ҙ localStorage дёӯпјҢдҪҶжҳҜеңЁ localStorage дјҡиҮӘеҠЁе°Ҷ localStorage иҪ¬жҚўжҲҗдёәеӯ—з¬ҰдёІеҪўејҸ
иҝҷдёӘж—¶еҖҷжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё JSON.stringify()иҝҷдёӘж–№жі•пјҢжқҘе°Ҷ JSON иҪ¬жҚўжҲҗдёә JSON еӯ—з¬ҰдёІ

        var data={
            name:'zhangsan',
            sex:'man',
        };
        var d=JSON.stringify(data);
        storage.setItem("data",d);
        //е°ҶJSONеӯ—з¬ҰдёІиҪ¬жҚўжҲҗдёәJSONеҜ№иұЎиҫ“еҮә
        var json=storage.getItem("data");
        var jsonObj=JSON.parse(json);

жҖқиҖғжҖҺд№ҲеҫҖ localStorage дёӯеӯҳе…Ҙж•°з»„пјҢе№¶дё”еҸҜд»Ҙеҗ‘ localStorage дёӯиҝҷдёӘж•°з»„дёӯж·»еҠ е…ғзҙ
