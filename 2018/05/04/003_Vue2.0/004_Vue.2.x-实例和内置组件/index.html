肆、Vue2.x-实例和内置组件

发表于2018-05-04|更新于2021-05-08|Vue2.x
|阅读量:

第1节：实例方法

）$mount方法

  • $mount方法是用来挂载我们的扩展的

）$destory()卸载方法

  • 用$destroy()进行卸载，我写了一个button按钮，点击后卸载整个挂载

）$nextTick()

  • 用于在create生命周期或者methods方法中也能获取到dom对象

第2节：实例事件（不常用）

  • 实例事件就是在构造器外部写一个调用构造器内部的方法。这样写的好处是可以通过这种写法在构造器外部调用构造器内部的数据。

  • 一、$on 在构造器外部添加事件。

  • 二、$once执行一次的事件

  • 三、$off关闭事件

第3节：内置组件 - slot（插槽）

  • slot是标签的内容扩展，也就是说你用slot就可以在自定义组件时传递给组件内容，组件接收内容并输出。
