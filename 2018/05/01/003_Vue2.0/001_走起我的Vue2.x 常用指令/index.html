其他内部指令

1.v-pre 在模板中跳过 vue 的编译，直接输出原始值。就是在标签中加入 v-pre 就不会输出 vue 中的 data 值了。

2.v-cloak 指令，在 vue 渲染完指定的整个 DOM 后才进行显示。它必须和 CSS 样式一起使用

3.v-once 指令，在第一次 DOM 时进行渲染，渲染完成后视为静态内容，跳出以后的渲染过程。
