初识组件

一、全局化注册组件

  • 全局化就是在构造器的外部用 Vue.component 来注册，我们注册现在就注册一个组件来体验一下

二、局部注册组件

  • 局部注册组件局部注册和全局注册组件是相对应的，局部注册的组件只能在组件注册的作用域进行使用，其他作用域使用无效。

三、组件和指令的区别

  • 组件注册是一个标签，而指令注册的是已有标签里的一个属性。在实际开发中我们还是用组件比较多，指令用的比较少。因为指令看起来封装的没那么好，这只是个人观点。

四、Component 组件 props 属性设置
