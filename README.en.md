> A personal blog built with [Hexo](https://hexo.io/) and the [Butterfly](https://butterfly.js.org/) theme.

## 💻 Setup

- Node.js 22+
- pnpm 11

## 🚀 Usage

```bash
pnpm install
pnpm run server
```

## 📦 Deploy

```bash
pnpm run clean
pnpm run build
pnpm run deploy
```
